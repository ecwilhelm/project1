LaTeX Tutorial
  • Introductory Page
  • Step-by-step explanation with sample code
  • The tutorial introduces novice users to the document markup language LaTeX, and enables them to create a LaTeX document using basic structural and formatting functions. The tutorial was built with HTML and CSS.
  • It uses an inverted pyramid approach, first describing the general structure and use of LaTeX commands, then moving on to structural elements like chapters, headings and paragraphs, and visual aspects like images and font styling. It concludes with a sections about incorporating citations and bibliographies.
  • Every section includes a step-by-step explanation with sample code, and an exercise with sample solutions. The exercises build upon each other and produce a fully functional and formatted document in the end. Staying true to LaTeX's cross-platform compatibility, ScribTeX, a web-based platform-independent editor is recommended for users who don't have a LaTeX installation on their computer.
  • Originally created as a stand-alone lesson as an assignment for an Instructional Design course at IIT, it was later combined with other students' tutorials to create The Research Paper Toolbox, an online course providing various tools and methods helping graduate students in the process of writing a research paper.
  • The interface is kept very simple, minimizing distraction for learners. The only decorative elements are the icon representing the lesson, and the colored border indicating which lesson the current pages belongs to.
